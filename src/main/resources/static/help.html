    Méthode: GET

    Méthode: GET

  • URL: https://rss22-ridel.cleverapps.io/rss22/resume/xml/{guid}

    Méthode: GET
    Résumé: Affiche l'article complet dont l'identifiant est {guid} au format XML

  • URL: https://rss22-ridel.cleverapps.io/rss22/resume/html/{guid}

    Méthode: GET
    Résumé: Affiche l'article complet dont l'identifiant est {guid} au format HTML

    Méthode: POST
    Résumé: Ajoute le flux rss22 décrit, Si le flux est déjà présent, une erreur est retournée. Sinon il est ajouté à la base de données.

  • URL: https://rss22-ridel.cleverapps.io/rss22/delete/{guid}

    Méthode: DELETE
    Résumé: Suppression de l'article dont l'identifiant est {guid}.
